Full Course: Mastering Microsoft Advanced Excel

Welcome to our comprehensive course on Microsoft Advanced Excel! This tutorial is designed for those who have already mastered basic Excel skills and are ready to delve deeper into its powerful features. Let's get started!

Module 1: Powerful Data Analysis Techniques

In this module, we will explore various data analysis techniques that will transform the way you work with your data in Excel. We will cover topics such as:

  • Data Validation
  • PivotTables and PivotCharts
  • Conditional Formatting
  • Advanced Filtering Techniques

Module 2: Macros, VBA, and Automation

Take your Excel skills to the next level by learning how to automate repetitive tasks using macros and VBA (Visual Basic for Applications). This module covers:

  • Creating, editing, and debugging macros
  • Understanding VBA syntax and programming concepts
  • Building custom Excel add-ins
  • Linking macros to buttons for easy access

Module 3: Advanced Formulas and Functions

Discover the power of advanced formulas and functions in Excel. In this module, we will cover:

  • Using array formulas and recursive formulas
  • Advanced conditional formatting with IF, IFERROR, and IFS functions
  • Working with lookup functions such as INDEX, MATCH, and XLOOKUP
  • Creating custom functions using Defined Names and Volatile Functions

Module 4: Modeling and What-If Analysis

Learn how to build robust models in Excel, including performing what-if analysis. This module covers:

  • Scenario Manager and Goal Seek
  • Data tables and one-way and two-way data tables
  • Sensitivity analysis with Solver add-in
  • Using Excel's Monte Carlo Simulation for risk analysis

Module 5: Creating Dashboards and Reports

Bring your data to life with interactive dashboards and professional reports. In this module, we will cover:

  • Creating sparklines and using them effectively
  • Designing compelling charts and graphs for different purposes
  • Using SmartArt to create visually appealing diagrams and infographics
  • Linking Excel files with PowerPoint and creating combined presentations

Module 6: Advanced Charting and Graphing Techniques

Take your charting skills to the next level by exploring advanced techniques such as:

  • Creating custom charts using built-in tools
  • Using Excel's Sparkline and Sunburst Chart
  • Creating 3D charts for visual impact
  • Integrating charts with Power BI for deeper analysis

Module 7: Working with Large Data Sets and External Data

Learn how to handle large data sets more efficiently, including importing and linking external data. This module covers:

  • Importing and linking data from various sources such as databases, text files, and websites
  • Using Excel's Power Query to clean, transform, and enrich your data
  • Creating Power Pivot models for faster analysis of large data sets
  • Optimizing Excel performance when working with large data sets

Module 8: Collaborating and Sharing Your Work

Learn how to collaborate and share your work effectively using Excel's built-in tools. This module covers:

  • Sharing a workbook on OneDrive or SharePoint
  • Using Excel's co-authoring features to collaborate in real time with others
  • Setting permissions and managing access to your workbooks
  • Publishing your workbook as a dashboard for easier consumption by others

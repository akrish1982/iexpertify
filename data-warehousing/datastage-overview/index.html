What is DataStage?

DataStage is an ETL tool which extracts data, transform and load data from source to the target. The data sources might include sequential files, indexed files, relational databases, external data sources, archives, enterprise applications, etc. DataStagefacilitates business analysis by providing quality data to help in gaining business intelligence.
